Скопировать репрозиторий на ПК :
	git clone ссылка на репрозитори
Создать файл:
	cat > имя файла
Проверить состояние рабочего каталога:
	git status
Добаление изменения:
	git add имя файла
Дополнительно проверим состояние рабочего каталога:
	git status
Добавление снимка состояния в проект:
	git commit -m "описываем что сделали"
Отправка измененных файлов в удаленные репрозитори:
	git push
